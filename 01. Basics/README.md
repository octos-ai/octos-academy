### Installation

Recomendamos que use anaconda para instalar python.
We highly recommend using anaconda for installing python. [Clique](https://www.anaconda.com/download/) para ir à página de donwload do Anaconda's. Certifique-se de baixar a versão Python 3.6.
Se você está em uma máquina windows:

- Abra o executável depois de terminar o download e siga as instruções.
- Depois de completa a instalação, abra `prompt Anaconda` a partir do menu inicial. Isso vai abrir um terminal com o python habilitado.

Se você está em uma máquina linux:

- Abra o terminal e navegue até o diretório para onde foi baixadoo Anaconda

- Mude a permissão do arquivo para que possa ser possível executá-lo. Se o nome do arquivo baixador for `Anaconda3-5.1.0-Linux-x86_64.sh`, então use o seguinte comando:

  `chmod a+x Anaconda3-5.1.0-Linux-x86_64.sh`

- Agora, rode o script de instalação `./Anaconda3-5.1.0-Linux-x86_64.sh` e siga as instruções de instalação no terminal.
